# 图片资源说明

本目录用于存放深圳市盛世德坤国际物流有限公司网站所需的图片资源。

## 建议添加的图片

1. **logistics-hero.jpg** - 网站首页的主视觉图片，建议使用高质量的国际物流相关图片，如飞机、货轮、卡车等元素的组合。

2. **company.jpg** - 公司简介部分使用的图片，可以是公司办公环境、团队合影或公司标志的高清图片。

3. **fba-services.jpg** - FBA线路服务页面的图片。

4. **ocean-freight.jpg** - 海运服务相关图片。

5. **air-freight.jpg** - 空运服务相关图片。

6. **warehousing.jpg** - 仓储服务相关图片。

7. **dg-cabinet.jpg** - 纯电DG柜服务相关图片。

8. **about-us.jpg** - 关于我们页面的图片。

## 图片规格建议

- 图片格式：JPG或PNG
- 分辨率：至少1920x1080像素
- 文件大小：单个图片建议不超过2MB
- 图片质量：70-80%的压缩率，保持视觉质量的同时优化加载速度

## 图片版权说明

请确保所有使用的图片都有合法的使用权，可以通过以下途径获取：
- 公司自有拍摄的图片
- 购买商业图库授权
- 使用免费可商用的图片网站资源

## 注意事项

- 请使用与公司品牌形象相符的专业图片
- 避免使用模糊、低质量的图片
- 确保图片内容与实际服务相符
- 图片添加后，请确保文件名与网站代码中的引用一致
